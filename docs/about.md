# 關於 UNC TSA

## 我們的使命

北卡羅來納大學台灣學生會（UNC TSA）致力於：
- 促進台灣文化在校園的傳播與交流
- 為台灣學生提供學術與生活上的支持
- 建立跨文化交流的平台
- 培養領導力與團隊合作精神

UNC TSA 成立於 [年份]，是北卡羅來納大學最具活力的學生組織之一。多年來，我們舉辦了眾多成功的活動，包括：

## 加入我們

我們歡迎所有對台灣文化感興趣的同學加入 TSA！無論是：

- 參與活動策劃
- 擔任幹部職務
- 提供建議與想法
- 參與文化交流

每個人的參與都能為我們的社群帶來獨特的價值。
